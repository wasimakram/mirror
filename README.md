# Mirror

Google Glass mirror API ruby wrapper.

## Installation

Add this line to your application's Gemfile:

    gem 'mirror'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mirror

## Usage

* Timeline
  - Card `insert`, `update`, `get`
  - CardOptions
  - CardBundle
* ShareEntities
* Subscriptions
